## 🚀 Instalación y Uso
1. Clonar el repositorio:
   ```bash
   git clone https://github.com/OptionIA/Gemini-AI-lens-2025/main.git
   ```
2. Instalar dependencias:
   ```bash
   pip install -r requirements.txt
   ```
3. Ejecutar el programa:
   ```bash
   python src/main.py
   ```
